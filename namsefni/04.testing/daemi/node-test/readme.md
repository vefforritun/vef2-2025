# node test dæmi

Til að keyra:

```bash
npm install
npm run test

# í "watch" mode
npm run watch
# eða þar sem þetta er byggt inn í node
node --test --watch

# Coverage
npm run coverage
# coverage er ennþá skilgreint sem "experimental"
node --test --experimental-test-coverage

# Hægt að gera bæði
node --test --watch --experimental-test-coverage
```
